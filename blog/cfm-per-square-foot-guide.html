CFM per square foot guide

Created 2026-07-16 · Updated 2026-07-16

Formula

CFM/ft² = CFM ÷ floor area

Use it carefully

This ratio compares airflow to area but does not include ceiling height or ventilation purpose.

FAQ

Is this a code requirement?

Not by itself; verify local and project requirements.
