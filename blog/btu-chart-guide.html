BTU chart guide

How to use an HVAC BTU chart

A conversion row is useful for comparing labels, not for replacing a load calculation.

Created 2026-07-16 · Updated 2026-07-16

Read the unit first

Check whether a chart reports Btu energy, BTU/h capacity, kW thermal capacity, or refrigeration tons. HVAC equipment rows normally use BTU/h.

Know the limits

Room-size charts are screening aids. Heat gain, envelope performance, moisture, solar exposure, and occupancy can change the actual load.

FAQ

Can a BTU chart select equipment?

No. It converts or compares capacity; final selection requires a suitable load calculation and manufacturer performance data.
