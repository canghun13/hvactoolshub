How to Size HVAC Ductwork

Start with verified airflow, then check area, velocity, pressure loss, fittings, and available sizes as one system.

Created 2026-07-16 · Updated 2026-07-16

Start with required airflow

Each run needs an assigned airflow. Room airflow and equipment design determine this value; a duct size alone does not establish it.

Convert airflow to face area

For a chosen velocity, area equals CFM divided by FPM. Select an available round or rectangular size, then calculate the actual velocity again.

Check the entire path

Straight duct is only one part of resistance. Include fittings, transitions, terminals, filters, dampers, and equipment external static pressure in the final design.

Use a documented design method

ASHRAE describes equal-friction and other duct design approaches; their selection depends on the system and project constraints. Final sizing should be reviewed by a qualified professional.

FAQ

Can one velocity fit every duct run?

No. Main ducts, branches, noise-sensitive spaces, and available geometry often need different criteria.
