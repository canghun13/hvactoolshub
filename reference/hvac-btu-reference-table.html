HVAC BTU reference table

Common thermal-capacity values for checking labels and changing units—not for selecting equipment by room size alone.

Created 2026-07-16 · Updated 2026-07-16

Formula and unit scope

kW thermal = BTU/h ÷ 3,412.141633
RT = BTU/h ÷ 12,000

All watt and kilowatt values in this table are thermal-capacity conversions. A product’s electrical input is a separate specification and must not be inferred from this table.

How to use a capacity row

  1. Match the row to a stated thermal capacity, not a floor-area rule.
  2. Use the kW or ton column only to express the same thermal rate in another unit.
  3. Check delivered capacity, airflow, and conditions in the manufacturer’s data before selection.
  4. Use a documented building-load calculation to decide whether the capacity is appropriate.

Example

A 48,000 BTU/h capacity is 4 RT or about 14.07 kW thermal. It does not specify the electrical circuit, power draw, humidity performance, or duct arrangement.

FAQ

Are table values equipment selections?

No. They are unit conversions and nominal capacity comparisons. Building load and manufacturer data determine whether equipment is suitable.
